[목적] OK금융그룹 HR 분석 대시보드 설계

[요구사항]
- 피벗 기반 필터 조합 가능 (직군, 직급, 계열사, 연령, 신분 등)
- PL/Non-PL 직군 분리, 관리자 여부 구분
- 아래 주요 영역을 포함한 대시보드 위젯 설계

[영역별 대시보드 구성]

1. 인력현황 표
- 기본 테이블 + 열: 계열사, 직급, 신분, 연령대
- 컬럼 비교 항목: 전월말 대비, 전년말 대비, 전년동기 대비

2. 입퇴사 추이 차트
- 일/주/월/누적 기준 선택 가능
- 아래 테이블: 입퇴사자 리스트 (이름, 직급, 나이, 입사년도, 경력, 퇴사사유)

3. 관리자 현황
- 관리자 리스트 카드: 사진, 직급, 직책, 평가요약, 경력경로
- Succession 후보 및 퇴사리스크 알림 포함

4. 생산성 지표
- KPI 카드형 위젯: HCROI, 인당생산성, 인건비 비율 등
- 연도별 라인 차트, 계열사 비교 막대형

5. AI 인사이트 요약
- 이상치 알림: 이직집중, 고령화, 고성과 저보상
- 예측 카드: 향후 인력감소 부서
- 대체인력 제안: 공석대응 제안

[UI 구성 제안]
- React + shadcn/ui + recharts 기반
- 대시보드 위젯 단위 구성 (Card, Table, Donut, Bar, Line)
- 상단 공통 필터 영역 포함 (Select, MultiSelect, DateRange)

[출력]
- 대시보드 UI 구조도
- 컴포넌트 분할 기준
- 각 위젯별 prop 및 예시 데이터 스키마
- 인터랙션 동선 흐름도
