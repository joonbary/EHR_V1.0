  🎯 디자인 철학 & 원칙

  1. 🎭 Modern Neumorphism + Glassmorphism

  /* 카드 디자인 */
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
  border-radius: 12px;
  backdrop-filter: blur(10px);

  /* Neumorphic Shadow */
  box-shadow: 8px 8px 16px rgba(0,0,0,0.08),
             -8px -8px 16px rgba(255,255,255,0.7);

  2. 🌊 Smooth Micro-Interactions

  /* 호버 효과 */
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
  transform: translateY(-2px); /* 카드 호버 */
  transform: translateX(5px);  /* 슬라이드 호버 */

  3. 🎨 Semantic Color Coding

  /* 우선순위별 색상 */
  .priority-critical: #6f42c1 (보라)
  .priority-high: #dc3545 (빨강)
  .priority-medium: #ffc107 (노랑)
  .priority-low: #28a745 (초록)

  /* 세션 유형별 색상 */
  .session-performance: #e7f3ff
  .session-leadership: #fff3cd
  .session-skill: #d4edda
  .session-career: #f8d7da

  🎵 애니메이션 & 전환 효과

  Easing Functions

  cubic-bezier(0.4, 0, 0.2, 1) /* Smooth */
  ease-in-out /* Standard */

  Animation Types

  /* 등장 애니메이션 */
  opacity: 0 → 1
  transform: translateY(20px) → translateY(0)

  /* 호버 애니메이션 */
  transform: translateY(-2px)
  box-shadow: 증가
