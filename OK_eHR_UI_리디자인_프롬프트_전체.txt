
📁 프로젝트명: OK금융그룹 e-HR UI 리디자인 프로젝트
🧭 목적: OK금융 e-HR 시스템 전반의 UI/UX를 디자인 시스템 기반으로 통일감 있고 현대적인 방식으로 재구축

🧱 UI 리디자인 설계 기준 (스타일가이드 요약)
1. 컬러 시스템
   - Primary: #1443FF, Accent: #00B7C2, Neutral: gray-100~900
2. 폰트: 'Noto Sans KR', base 16px, heading 24~32px
3. 여백: 8px 단위 grid 시스템 적용
4. 아이콘: 단색, Lucide 기반 / 활성 시 Primary 색 강조
5. 카드/위젯: 통일된 radius, shadow, padding, delta 스타일 (녹색/빨강)
6. 다크모드: Tailwind dark: 기반 전역 지원 및 ThemeToggle 컴포넌트 사용
7. 중복/인라인 스타일 제거 및 컴포넌트 재사용화

🔧 MCP 서버 지시사항
- 모든 단계는 Shrimp Task Manager를 통해 순차 실행
- 단계별 진행률을 WebSocket으로 전송 (Shrimp 연동)
