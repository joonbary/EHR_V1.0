/sc:improve -magic -persona-frontend

🎯 목적: 직원 AI 분석 상세 팝업의 UI/UX 고도화 및 정보 밀도 강화

🔧 작업 범위:
1. **8대 핵심역량 레이더 차트 개선**
   - 기존 레이더 차트에 `전사 평균` 대비 점수선 추가 (회색 dashed line)
   - `동직군 평균`, `본인 점수` 전환 가능한 탭형 비교 기능
   - 각 항목별 `상승/하락 추이` 표시 (아이콘: 📈📉)

2. **강점/약점 시각화 강화**
   - 강점 TOP3 외에 전체 강점 키워드 (최대 5개) 태그로 표시
   - 개발 필요 역량 3개까지 표시 (순위 및 색상 구분: 오렌지→빨강)
   - 마우스 오버 시 관련 피드백 요약 툴팁 표시

3. **AI 종합 피드백 구조화**
   - “요약 보기”/“상세 보기” 토글 버튼
   - 요약: 한 문장 요약
   - 상세: 장점 요약 + 보완점 요약 + AI 자신도 평가 수치 (`AI 신뢰도` 퍼센트)

4. **추천 경력 및 교육방향 개선**
   - 추천 경력방향을 카드형으로 시각화 (아이콘: 🧭 프로젝트관리 / 🎯 팀리더십)
   - 내부 경력자 유사 패턴 기반 진단 문구 예시 추가
   - 교육 추천은 카테고리별 탭 구성 (리더십 / 전략 / 변화관리 / 커뮤니케이션 등)

5. **하단 액션버튼 기능 추가**
   - 📎 PDF 저장 버튼
   - 📊 전사 내 백분위 순위 보기
   - 👥 유사 인재 프로필과 비교 보기 (동일 직무 or 등급 1위와 비교)
   - 🤖 GPT 분석 근거 보기 (AI 판단의 요약 기반 내용 팝업)

💡 기타 UI 디테일:
- 점수 컬러 스케일: 0~100 → 빨강~파랑 스펙트럼
- 강점 키워드는 강조 색상 (primary color)
- 모든 차트 반응형 적용 및 모바일 최적화

🧪 테스트 시나리오 제안:
- 다양한 점수 분포 (예: 강점 위주 vs 약점 위주 vs 평범한 분포)
- PDF 저장 → 항목 누락 없는지 확인
- “GPT 분석 근거 보기” → 에러 없는 자연어 표현 확인
